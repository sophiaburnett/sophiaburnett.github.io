[**The Human Inside Project**](https://sites.google.com/view/humaninside/home) Part 1: I'm not a robot, is a citizen science project that will rely on crowdsourced data. Please do participate, it only takes roughly 15 to complete. Use the form below for English, or go via the [project site](https://sites.google.com/view/humaninside/home) if you would like French. This is an ongoing collaboration, and Mandarin and Vietnamese will soon be available.
If you are interested in working on the project, please do get in touch.
